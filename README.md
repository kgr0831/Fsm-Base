# FSMBase for Unity

이 코드는 Unity에서 사용할 수 있는 간단한 FSM(유한 상태 기계) 베이스 클래스입니다.  
불필요한 복잡성을 제거하고, 최소한의 코드로 상태 전환과 관리가 가능하도록 설계되었습니다.  
AI, 플레이어 동작 제어, UI 로직 등 다양한 상황에서 활용할 수 있습니다.

## 주요 특징

- 단순한 구조와 빠른 성능
- 상태 정의를 위한 인터페이스(IState) 제공
- 조건 함수를 통한 자동 상태 전이
- 다음과 같은 기본 유틸리티 포함  
  - `Set(IState)` : 현재 상태를 변경  
  - `Add(IState, IState, Func<bool>)` : 전이 조건 추가  
  - `Is(IState)` / `Is<T>()` : 현재 상태 확인  

## 사용 방법

FSMBase를 상속받은 클래스에서 상태 객체를 생성하고,  
`Set()`으로 초기 상태를 지정한 뒤 `Add()`로 상태 전이 조건을 정의하면 됩니다.  
상태는 `IState` 인터페이스를 구현하여 만들 수 있습니다.  

Unity 실행 중 조건이 충족되면 자동으로 상태가 전환됩니다.
